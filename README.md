The Smart Surveillance System is a Python-based application with computer vision techniques to provide real-time object detections using AI in the background. In particular, the application is focused on detecting objects and people from video feeds in order to make the process of intelligent surveillance feasible in real-time. The main scope of work consists of automation of the video analysis, real-time detections, screenshots capturing, and evaluating performance with the use of accuracy, fps etc. The project can be applied for different purposes – from smart security monitoring or campus surveillance to traffic observing and automatic intruder detection. It is also important that a modular structure makes it easy-to-use, scalable, and reliable.\n\nThe whole project has several modules where files execute certain responsibilities: `main.py` – defines the execution order; `config.py` – provides configurable parameters (thresholds, paths etc); `model_loader.py` – provides loading the pretrained YOLOv8 deep learning model; `detector.py` – executes the detection algorithm itself (frame capturing, inference, bounding box creation, and labels highlighting); `ui.py` – responsible for the interface display and interaction with users (object visualization); `evaluation.py` – contains metrics calculating methods for further analysis of system performance.
